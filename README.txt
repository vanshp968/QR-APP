1. QR Code Encoder/Decoder App
   This Python app encodes and decodes QR codes using the qrcode and opencv-python libraries.
   **How to Run:**
   - Make sure you have Docker installed on your machine.
   - Build the Docker image: `docker build -t qr-code-app .`
   - Run the Docker container: `docker run -p 4000:80 qr-code-app`
   - Access the app in your browser: http://localhost:4000
